Usage: ./my_hunter

Description:
  A simple hunting game where you use the
  mouse to shoot targets and the 'Escape' key to access the pause menu.

Options:
  -h,       Display this help message and exit.

Game Controls:
  - Mouse Click:   Use the left mouse button to shoot targets.
  - Left and Right Arrow: Move in the setting.
  - A and S:       Add or Subtract to the volume of the music.
  - Esc:           Exit the setting and go back to the base menu.
